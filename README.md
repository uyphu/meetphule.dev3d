## 🧪 Getting Started

### 1. Clone the Repo

```bash
git clone https://github.com/uyphu/meetphule.dev3d.git
cd meetphule.dev3d
```
### 2. Install Dependencies
```bash
npm install
```
### 3. Run the Dev Server
```bash
npm run dev
Visit http://localhost:5173
```
### 🌐 Deployment
Easily deploy to Vercel using their CLI or GitHub integration.

```bash
npm run build
```
